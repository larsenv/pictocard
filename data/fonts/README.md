# Fonts

Place custom font files (`.ttf`, `.otf`, etc.) in this directory.

The following font families are supported out of the box (filenames are case-sensitive):

| Display Name   | Family       | Expected file          |
|----------------|--------------|------------------------|
| Digital 7      | digital7     | digital7.ttf           |
| Humming        | humming      | humming.ttf            |
| Nintendo Labo  | nintendolabo | nintendolabo.ttf       |
| Nintendo U     | nintendou    | nintendou.ttf          |
| NSMFU          | nsmfu        | nsmfu.ttf              |
| Pop Happiness  | pophappiness | pophappiness.ttf       |
| Rodin NTLG     | RodinNTLG    | RodinNTLG.ttf          |
| Seurat         | seurat       | seurat.ttf             |
| SMMEX          | smmex        | smmex.ttf              |
| Splatfont      | splatfont    | splatfont.ttf          |
| Splatfont 2    | splatfont2   | splatfont2.ttf         |
| WarioWare      | warioware    | warioware.ttf          |

Fonts are sourced from the [RiiTag-Next](https://github.com/PretendoNetwork/RiiTag-Next) project.
If a font file is missing, the card generator will fall back to the system sans-serif font.
